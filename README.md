## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-github-profile/portfolio.git
    ```

2. Navigate to the project directory:
    ```bash
    cd portfolio
    ```

3. Install dependencies (If Needed):
    ```bash
    npm install
    ```
